# LockNote（桌面端）

LockNote 是一款**简单、可靠、离线优先**的桌面加密笔记软件。

它将你的笔记内容在本地使用现代密码学进行加密保存：
- 内容使用 **AES-256-GCM** 加密
- 密钥由你的密码通过 **Argon2id** 派生

**无账号、无云端、无联网依赖**：你的数据只在你的设备上。

## 适合谁

- 希望笔记默认加密、对安全有要求、数据归自己所有的人
- 经常离线使用（出差/内网/无网环境）的人
- 想要长期可读、可备份、可迁移的本地笔记的人

## 核心特性

- **加密存储**
  - 笔记标题与内容加密后落盘
  - SQLite 仅保存元数据，不保存明文内容
- **离线优先**
  - 完全本地使用，不依赖网络
- **启动解锁**
  - 未解锁前不可访问任何笔记
  - 支持手动锁定
- **Markdown 编辑**
  - 支持 Markdown 语法与预览
- **标签管理**
  - 为笔记添加标签，便于分类与筛选
- **全文搜索**
  - 搜索标题、内容与标签
- **历史版本**
  - 自动保存历史版本，支持回滚
- **备份与恢复**
  - 支持手动备份与恢复

## 安全说明（摘要）

- 解锁后主密钥仅驻留内存
- 每篇笔记使用独立随机 `nonce`
- 密文文件采用原子写入，尽量避免中途异常导致数据损坏

> 如果你遗失了登录密码且同时遗失恢复密钥，数据将无法恢复。

## 技术栈

- 后端：Go + Wails v2
- 前端：React + TypeScript + TailwindCSS
- 存储：SQLite（元数据）+ 本地密文文件（内容）
- 加密：AES-256-GCM + Argon2id

## 快速开始（开发者）

### 环境要求

- Go 1.21+
- Node.js 18+
- Wails CLI v2

### 安装 Wails CLI

```bash
go install github.com/wailsapp/wails/v2/cmd/wails@latest
```

### 开发运行

```bash
cd frontend && npm install && cd ..
wails dev
```

### 构建

```bash
wails build
```

如需多平台构建/带参数构建，可参考项目根目录的 `build.md` 与 `build.sh`。
